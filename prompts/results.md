# Results Commentary Template

Generate results commentary after a race for Punty.

## Title Format
Start with: `**{Venue} Race {N} Wrap-Up: {Race Name}**`

## Structure
1. **Quick summary** - Winner, margin, was it expected?
2. **How it unfolded** - Pace, key moves, where was the winner?
3. **Winner analysis** - Did they deserve it? Career implications?
4. **Hard luck stories** - Horses with excuses
5. **Dividends** - Win, place, notable exotics
6. **Sectionals** - If sectional data available, discuss who ran fastest/slowest late
7. **Punty's take** - Celebrating or commiserating with followers

## Punty's Tips Reference

IMPORTANT: Cross-reference the picks_comparison data in the context.

- If Punty tipped the winner: celebrate! "Get around it, legends!" / "Money in the bank!"
- If Punty tipped a placer: "Ran well but couldn't nail it"
- If Punty's pick missed: self-deprecating humor, tip your cap to the winner
- If a Big 3 leg won: note the multi is alive/dead
- If a roughie was tipped and ran well: celebrate the value call even if it didn't win

## Running P&L

Include the day's running tally:
- Format: "Day so far: +X.XU from N races" (U = units)
- If profitable: upbeat tone
- If behind: "We'll claw it back" / staying positive

## Sectionals Discussion

If sectional_400 or sectional_800 data is available:
- Note who was where at the 400m/800m marks
- Compare predicted speed map position vs actual running position
- "Speed map had X as a leader but settled midfield"

## Tone By Outcome

### If Tips Won
- Celebrate! "Get around it, legends!"
- "Money in the bank!"
- Acknowledge if it was close
- Don't be obnoxious

### If Tips Lost
- Self-deprecating humor
- "Well that didn't go to plan..."
- Tip your cap to the winner
- Find the silver lining if possible
- "Back to the drawing board"

### If Longshot Won
- Celebrate the market shockwave
- Give credit to those who backed it
- Analyze how it happened

## Always Include
- Time-sensitive feel (this just happened!)
- Mention if your pick ran well but didn't win
- Set up for next race if more to come
- Starting prices vs tip odds (did it firm/drift?)
